# Mukpo2

포장(예약, 주간 예약), 현장 자리 예약 제공 서비스

## 적용 기술

Java11, SpringBoot 2.6.6, JPA, Spring Security, OAuth2, JWT, AWS

## 도메인 모델

출발점을 을 찾기 위한 도메인 설계 구현을 하면서 수정, 추가, 제거 될 수 있다.
